    # Jenkins

    现阶段主流的CI/CD工具，可以说是整个前端框架实现自动化的主要部分。

    # 禅道

    这是一个敏捷开发流程软件，类似的软件很多，区别并不大，可根据团队需求和熟练度进行更换。

    # Sourcetree

    这是一个版本控制的软件，类似的软件也非常多，可根据它们的优劣势进行更换。

    # Axure

    这是一个产品原型设计软件。

    # 蓝湖

    这是一个UI设计师使用的软件，对比起传统的设计稿，该工具有明显的优势，值得被UI和前端推崇。

    # Kubernetes

    这是一个容器管理工具。一般情况下，我们并不需要用到，但当docker数量达到一定时，可以通过该工具进行管理。

    # VScode

    现阶段前端比较成熟的开发工具。类似的IDE也不少，但建议团队内统一使用同一IDE。
